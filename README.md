# SMART_LIGHTING_USING_LDR_WITH_LCD_DISPLAY
Smart lighting using LDR with LCD display is a real time application useful for monitoring light intensity and control the light based on light intensity. 

## 📖 Project Overview

The Smart Lighting System Using LDR with LCD display is an advancement in previous embedded systems project. it is designed in tinkercad which automatically control lighting based on ambient light conditions. The system utilizes a Light Dependent Resistor (LDR) to sense environmental light intensity and intelligently switch a lighting device ON or OFF and prints the state of the system on LCD display. This automation helps reduce energy consumption and improves lighting efficiency and user friendly.

## ⚙️ Working Principle

The system continuously monitors ambient light intensity using an LDR sensor. The sensor's resistance changes according to the amount of light falling on it.

- During daylight conditions, the system keeps the light source OFF.
- During low-light or dark conditions, the system automatically turns the light source ON.
- Display the state of the light bulb on LCD display, whether it is ON or OFF.
